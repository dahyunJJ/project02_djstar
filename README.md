# project02_djstar
> 사용자 편의성에 기반하여 UI/UX를 개선한 반응형 웹사이트 구현 <br>
> **기존 대전시민천문대 웹사이트** https://djstar.kr/

## 프로젝트의 목적
> 1. 기존 웹사이트를 분석하고 장점과 단점을 파악한 후, 벤치마킹 요소와 결합하여 리뉴얼 <br>
> 2. 복잡한 사이트맵 재구성 및 예약 프로세스 개선
> 3. 날씨 API를 활용하여 사용자에게 정보 제공

## 프로젝트 내 역할
> 분석 및 기획 : 50% <br>
> 발표자료(ppt) 작성 : 70% <br>
> 디자인 : 100% <br>
> 코딩 : 100% <br>
